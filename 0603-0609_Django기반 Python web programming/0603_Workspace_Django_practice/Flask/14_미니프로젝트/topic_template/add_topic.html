부들이란?


범죄자에게 범죄에 비해 너무 가벼운 처벌이 주어져 분노했던 경험이 있으십니까?
저희 부들(BudL)은 부당한 사법 판결에 의해 분노하였던 분들을 위해
더 나은 사회로 나아갈 수 있는 징검다리의 역할을 목표로 만들어진
대한민국 최초의 판결 평가 시스템입니다.

"BudL"은 세간을 떠들썩하게 만든 주요 범죄 사건들에 법정이 구형한 형벌 데이터를 수집합니다.
이후 판결문을 향한 SNS 사용자들의 반응을 수집 후 판결의 공명정대성을 평가합니다.
"BudL"은 사회 흐름에 맞는 법정 개혁을 위한 중요한 지표로써의 역할을 할 것입니다.
